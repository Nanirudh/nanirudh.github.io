A lightweight multi-agent research system that improves answer quality by combining question clarification, partial source visibility, independent reasoning, skepticism, and synthesis.

The project is intentionally built with plain Python (no agent frameworks) to make multi-agent LLM behavior easy to understand, debug, and extend.

*Github repo URL*: [Multi-Agent-Research-Assistant](https://github.com/Nanirudh/Multi-Agent-Research-Assistant)
